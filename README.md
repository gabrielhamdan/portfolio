# Portfolio

This is my personal portfolio built with React. It showcases my projects, skills, and multilingual content (PT/EN/FR).  

## Features

- Multilingual support using React Context
- Clean, modern UI built with React and CSS  

## Stack

- React  
- TypeScript  
- CSS / Flexbox & Grid  
- Axios (for fetching GitHub projects)  

## Usage

```bash
git clone https://github.com/gabrielhamdan/portfolio.git
cd portfolio
npm install
npm run dev
